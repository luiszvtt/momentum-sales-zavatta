Momentum Sales
Sistema de performance para profissionais de vendas autônomos.

Desenvolvi este projeto para resolver um problema real do meu dia a dia como corretor de imóveis: a falta de previsibilidade de renda e dificuldade em manter consistência.

Sobre:
O Momentum Sales transforma esforço diário em progresso visível e ajuda a responder:

Estou fazendo o suficiente hoje?
Quanto isso pode gerar de receita?
O que devo fazer agora?
Funcionalidades:
Dashboard com métricas diárias
Progresso de atividades (contatos, follow-ups, visitas, propostas)
Estimativa de receita em tempo real
Metas diárias e mensais
Controle de consistência (streak)
Feedback e direcionamento
Sistema de lembretes
Conceito:
Esforço → Progresso → Receita

Tecnologias:
React / Next.js
TypeScript
Firebase
Tailwind CSS
Objetivo:
Criar um sistema que transforme consistência em resultado previsível para profissionais de vendas.

Autor:
Projeto desenvolvido para resolver um problema real do meu próprio trabalho.
